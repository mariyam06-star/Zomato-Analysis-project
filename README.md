Project Overview :
This project is an interactive Power BI dashboard built using Zomato restaurant data. The dashboard provides insights into restaurant distribution, customer ratings, cuisines, online delivery availability, and table booking trends across different cities and countries.

The goal of this project is to transform raw restaurant data into meaningful visual insights for better business analysis and decision-making.

Dashboard Features :
Key Performance Indicators (KPIs)
Total Restaurants: 8.652K
Total Cities: 43
Countries Covered: 1
Average Rating: 2.77
Total Votes: 871

Visualizations Included :
1. Country-wise Restaurant Distribution
Interactive map visualization showing restaurant locations.
Helps identify restaurant concentration across regions.
2. Cuisine Analysis
Displays:
Count of cuisines
Average ratings by cuisine
Helps understand popular and highly rated cuisines.
3. Online Delivery Analysis
Pie chart showing restaurants offering online delivery.
Compares delivery-enabled vs non-delivery restaurants.
4. Table Booking Analysis
Donut chart displaying table booking availability.
Useful for analyzing customer service features.
5. Top Cuisines by Restaurant Count
Bar chart representing the most common cuisines.
Highlights customer food preferences and market trends.
6. Dynamic Filters
Country filter
Year filter
Interactive slicers for better data exploration
